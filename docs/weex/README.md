# Works

## [晩御飯を決めるアプリ（非公開）](https://github.com/yumchaa/gohan)

アプリの部分（Weex）の開発を担当

決定木モデルを使い、晩御飯の属性（色や場所や時間など）をアプリ上に表示し、ユーザはその属性を選択する  
最終的に選んだ属性から導き出された晩御飯をアプリ上に表示する（アキネイターみたいなもの）

機械学習モデルに訓練させるデータセット不足により、満足のいく結果が出せなかった

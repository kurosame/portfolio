# Summary

2018/04〜

用語の一覧と検索機能がある簡易アプリ

- Material Design
- Google 認証（Firebase Authentication）
- DB は Firebase（Storage にアップロードしたファイルを Functions で処理して、Firestore に保存している）
- Web プッシュ通知（Firebase Cloud Messaging + Service Worker）
